A responsive web application providing comprehensive information about bus parks, transport routes, fare estimates, and union contacts across Nigeria.

## 🌟 Features

- **Interactive Map Integration** - Locate bus terminals using Leaflet.js
- **Smart Search** - Find parks by name, city, or state
- **Route Explorer** - Browse routes with fare estimates and duration
- **Union Contacts** - Direct access to transport union information
- **Mobile Responsive** - Optimized for all devices
- **Offline Capable** - Client-side data storage

## 🚀 Live Demo

**URL**: [https://Michaelboluwatife.github.io/NaijaBus](https://Michaelboluwatife.github.io/NaijaBus)

## 📋 Pages

1. **Home** - Overview, search, featured parks & popular routes
2. **Bus Parks** - Directory with filtering by location
3. **Routes** - Route listings with fare information
4. **Details** - Individual park/route information with maps

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3 (Tailwind CSS), JavaScript (ES6+)
- **Maps**: Leaflet.js with OpenStreetMap
- **Icons**: Lucide Icons
- **Fonts**: Google Fonts (Inter)
- **Deployment**: GitHub Pages
